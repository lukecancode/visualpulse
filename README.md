# VisualPulse

VisualPulse is a Windows automation tool that monitors visual changes on screen and reacts instantly with automated clicks.

Instead of relying on scripts, APIs, or window titles, VisualPulse works purely by detecting pixel changes in a selected screen area.

## Key features

- Visual (pixel-based) event detection
- Instant reaction with human-like clicks
- Works even when the mouse is on another monitor
- No scripts, no coding required
